Taylor series:

 How to use:
This WebApp illustrate how Taylor series expansion can be used to approximate a function.
Select the domain you want to play (1D or 2D), the function you want to approximate and the degree of the approximating polynomial.
You will see the plot of the function and the approximation. Move the mouse over the canvas to move the center point of the expansion. Mouse scroll the canvas to zoom.

Press OK to see the function of each element in the WebApp.
